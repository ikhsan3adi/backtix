### システム要件

インストールを開始する前に、システムが次の要件を満たしていることを確認してください。

-   node.js v18以降
-   postgreSql 15以上
-   Redis V5以上

### インストール手順

1.  インストール手順に従ってください[バックエンドサービス](api-service.md)あらかじめ

2.  セットアップ環境

    名前を変更します`.env.example`に`.env`

    ホストとポート /オリジンの値はAPIサービスとは異なる必要があります

### 実行方法

1.  建てる

```bash
npm run admin:build
```

2.  アプリを実行します

```bash
npm run admin:start
```

-   または開発モード

```bash
npm run admin:dev
```
